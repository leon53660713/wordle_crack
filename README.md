Breaking Wordle Game
the goal is to break a game called "Wordle"

goal : guess a five-letter word

how to play :
start by guessing any five-letter word
based on your guess, the game will provide feedback
use the feedback to continue making guesses
repeat the process until you guess the correct word

遊戲流程 ：
先隨便猜一個字

然後接續繼續猜測
